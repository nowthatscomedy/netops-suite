# 장비 점검 프로파일 참고 예시

이 폴더의 기본 벤더 프로파일은 기존 Inspector 런타임에서 생성한 명령/파싱 규칙입니다.
`reference__*.yaml` 파일은 운영용 정답이 아니라, 사용자가 자기 장비에 맞게 복사해서 수정할 수 있는 참고용 예시입니다.

주의:

- 실제 장비 모델, OS, OS 버전에 따라 명령어와 출력 줄/값 위치가 달라질 수 있습니다.
- 참고용 프로파일을 그대로 운영 반영하지 말고, 장비 출력 예시를 붙여 넣어 미리보기 결과를 확인한 뒤 저장하세요.
- 정규식이나 YAML로 표현하기 어려운 값은 `custom_parsers.example.py`를 참고해 Python `parsing_*` 함수를 작성하세요.
